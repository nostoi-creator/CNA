Storylineストーリー

西暦が3000年にて終焉を迎え、世界暦が旧世界紀（オールドエイジ）から現世界紀（ニューエイジ）に突入した今、神は神ではいられず、また神秘はその全てが論理へと変わった。信仰無き世界で「分からないもの」とは「知らないもの」であり、もはや「フィクション」ですらなくなった。

魔法、神性、悪魔、天使。全てが説明"できてしまう"のであれば、それらに価値は無い。価値とは「今使えるもの」か否か──あるいは、「戦える」か否か。

人類の境が知性に限られた今、人は次の星を目指さんとする。先んじて地球の衛星たる月を拠とした『アウトサイダー』──残された星を掌握せんと地下を拠とした『インサイダー』──そして。

そのどちらでもなく、だが戦い続ける『リミナル』

汚染された大地。巨大な陰謀が渦巻き、国家の失われた世界では、より小さく無数にある規範が闘争に駆ける。過去と未来の消失、アイデンティティの喪失、イデオロギーの無意味化。人は今、『現在』だけを頼りにしていた。

それら全ての戦いが、一人の思索から始まったのだ。

旧性素粒子
Fusion Elementary Particle — F:EP
フュージョン・エレメンタリー・パーティクル
旧世界の科学においては単に『素粒子』や『基本粒子』と呼ばれた、当時の物質の最小単位。現世界紀に突入した現在ではその位置を『現性素粒子』へと譲り、旧世界紀の素粒子は『旧性素粒子』と呼ばれるようになった。"世界が壊れる"前の物理法則の象徴でもあり、そのことから『融合（フュージョン）』の名も飾る。言えばそれは『第四の壁』であり、その下限を見破った◆◆◆博士は世界がフラクタル構造であることを示唆した。

曰く、フィクションとノンフィクションの「境界殺し」である。夢物語だったはずのフィクションまで、人類は手を伸ばしてしまった。今や人類は幻想を作るすべを持たず、幻想を殺すすべばかりを持ち、幻想を踏み越えて今ある論理を振りかざす。少なくともそうなる前の方が平和だった。

アイデンティティの喪失。「今あるものが全て」。そうして人は恣意を抱くことを迫られた。

現性素粒子
Source Elementary Particle — S:EP
ソース・エレメンタリー・パーティクル
「大は小を兼ねる」「だが小とは大である」
発言者「◆◆◆」より
旧性素粒子を構成する、更にミクロな領域の「素粒子の中の素粒子」。旧世界の物理法則ではエネルギー利用は以ての外、本来観測不可能な領域だったが◆◆◆博士が発見"してしまった"。現世界においては主に核分裂による莫大なエネルギー源となっており、無尽蔵のエネルギーが旧世界にて実現困難であったもの、あるいは誇大妄想の域だったものを実現可能にまで引き上げた。

しかし既存技術自体は未だ追いついておらず、旧物理法則の見解においては現性素粒子は「分裂より融合の方が容易い」にも関わらず、技術者は核分裂を主要なエネルギーとするといった、歪で突飛な法則観が保たれたままだ。

現性素粒子の発見、その前後で世界の旧/現は区別される。旧世界における紀元前後（西暦）と同様である。一般的には粒子発見以前をオールドエイジ（OA）以後はニューエイジ（NA）と呼称する。なお西暦終焉はおおよそ3000年であると推察されるが、粒子発見後の時軸の不安定化が詳細の追究を遮断している為、不明瞭である。

プライマル・フォールアウト
Primal Fallout — P.F.
エネルギー利用を目的としてF:EPを核分裂させる際、低精度から漏洩したS:EPの特定ベクトル（エントロン/カオス子）が周囲のF:EPへと伝播する。するとエントロンの誘導により、強い衝突を受けたF:EPは放射性物質へと変異してしまう。それは旧世紀で核兵器と呼ばれたものの遥かに上をゆく拡散性を誇り、分裂エネルギーは非常にリスクが高いエネルギー源となる。誘導された放射性物質はよりマクロに移ろいでゆき、これが次第に沈下（フォールアウト）する。

通常のフォールアウトと異なるのは気流等の影響を受けない程度のミクロな範囲であること。二次的な汚染力は低いものの、強力な一次拡散は予め施された除去設備でなければ現世紀技術でも浄化は容易には行えない。またPFはこの通り通常の放射能汚染とは位相が異なるため、旧世紀従来の耐放射能装備では生半に防ぐことも叶わない点にある。故の『プライマル（Primal）』である。

一方、分裂自体を発生させなければS:EPの抽出は行えず、必然的に汚染と高エネルギー発生が伴う。現世紀の兵装の種別にある『ビーム』の系統は殆どがこのエネルギーを含んだものであり、甚大な被害を齎すと同時に、旧世界では物理的にあり得ないほどのエネルギーないしF:EPの投射が行われているのである。『レーザー』の系統はこれとは異なり、既に分裂したF:EP内部にあるS:EP、その特定カテゴリを純化させたクリーンエネルギーが『レーザー』に相当する。

旧性因子
IRREGULAR
今の理屈の「外」の存在。唯一今の世界で解析できず、なおここに存在するもの。論理で開拓された現世紀において『伝説』という存在を担保し、だからまだ世界から伝説は消えていない。

観測不可能性。模された現性因子が『１（有数）』であれば、これは真正の『０（無数）』、即ち可能性である。ただ己が感覚のみで突き進み、障害（境界）を殺し続けるだけなのだろう。例え先が見えずとも、それが古い時代の在り方だったから。
現性因子
DOMINANT
所謂『強化人間（ドミナント）』『ニュータイプ』等々、常人を凌駕する機体制御技術を持った存在に施された「旧性因子を模した」人工因子。最初に『イレギュラー』の研究を始めたのは『レイヴンズネスト』であり、現性素粒子の特定カテゴリーを人体に大量注入し制御インプラントを内蔵することで『イレギュラー』に近付こうとした。

その本質は「受動」であり、旧性因子の本質であるとされる「能動」には追いつかない。受容者は他者の精神への感受性が異様に高くなり殺気などの感知に優れるが、一方で心が壊れた者も多く、狂気にも至れず錯乱しただけだった。

現世紀の発端。ないし、旧世紀との「境界」そのものとなった人物。彼の現性素粒子発見が全てを変え、あるいは全てを作り、全てを壊した。現性科学の常識としてその存在のみが伝えられているが出生地はおろか名前すらまともに認識されておらず、ただ「世界の外へ消えた」とだけ考えられる。

REGULATION 利用規約
当オープンチャット（以下OC）はなりきりを主目的とします。利用者当人が持つ著作権の及ばない他の著作物・商標作品は、いかなる二次的商業化も認可致しません。

当OCは個人利用の範疇でクローズドコミュニティとして交遊することを主軸とするため、利用者のコミュニティ外部における問題行動・発言・違法行為に関する責任は当事者が負うものとします。また管理側に関する責任はコミュニティ統括者（OC管理人）が全責任を負い、利用者に負わせません。

当OCの利用規約はLINEオープンチャット利用規約、日本国法律に基づいています。管理・副管理権限を所有する役職者はOC「Cosmos New Age」（以下CNA）内、またその関連コミュニティにてその権限の及ぶ範囲でのみ強制力を有しますが、コミュニティ外部ではその限りではありません。
CHARACTER キャラクター利用について
_₁_

当OCは全也（クロスオーバー）にカテゴリされます。一般利用者様のキャラクターの掛け持ちにつきましてはOC管理の負担の軽減、またOC全体の質向上のため不可とさせていただきます。

折伽羅（オリジナルキャラクター）の利用は可能です。ただしこれは「版権の二次創作」ではないことを条件と致します。版権二次創作に関しては一次著作物のキャラクターの利用者様との兼ね合いを考慮して不可と致します。

＊ 版権 著作権。当OCにおいては出典元が明示されたキャラクターを版権として扱います。商業作品の是非ではなく、著作権が利用者様に帰属するオリジナルキャラクターであっても版権キャラクターとして扱います。『CNA』の世界観に沿ってキャラクター設定が一部改変されていたとしても、その著作権は全て出典元にあります。
＊ 折 作品の出典元が存在しないもの。当OCにおいては外部引用ではないキャラクター、『CNA』世界観に沿って新たに制作されたものを「折」ないしオリジナルキャラクターとして扱います。その著作権は利用者様に帰属するため、コミュニティ内外部で設定の改変・再利用を管理が制限することはありません。
_₂_

当OCはなりきり（也／ロールプレイ）を主目的とするため、なりきりとしての過激な発言を認可しますが、ロールプレイ外（炉留外／背後部屋）での差別発言や中傷行為は注意喚起を促すことがございます。またなりきり中につきましても当OCの媒体となるLINEオープンチャット自体のプライバシーポリシーに抵触する文言はOC自体の存続に関わりますので厳重にご注意ください。

背後部屋（ロール外）における半也（キャラクターを噛ませた利用者としての状態）は許可致します。ただし半也は全ての利用者様に理解されるわけではなく、コミュニティ内での問題に繋がりかねないことを念頭に入れてください。半也は能動的行為であり、原則としてそこから生じる問題自体は利用者様自身の責任となります。

_₃_

伽羅の着席時にはOC管理にその旨を申請していただきます。業務連絡トークルームにて申請後、キャラクター名簿のコメント欄に出典元（折も同様）を記載してください。

¹業務連絡にてOC管理をメンションして申請
²# Character ノートコメント欄に記載

この二点をもって伽羅の枠取りになります。二点を行われていない利用者様につきましては、枠取りが完遂されていないと見なしますのでご注意ください。なおOC内でのロール行動の許可自体は以下を完遂後に致します。

³業務連絡にて所属・役職を申請（機体所有者は申請後 # Machine コメント欄に記載）
⁴# Position 各ノートのコメント欄に記載

版権利用者様の『機体』につきましては、原則として原作キャラクターが有する物のみが利用可能になります。

例 ARMORED CORE Ⅵ より ラスティ
　↳ スティールヘイズ 及び スティールヘイズ・オルトゥス

折伽羅利用者様の『機体』に関しては、CNA世界観設定にある既存の版権機種に倣った上で制作を行ってください。折伽羅による新規の機種は世界観の統一性・バランスの観点から基本的には認可しません。ただし全体の品質向上にあたってOC管理に相談していただければ一考の機会になります。

_₄_

当OCは世界観重視の運営を行います。その際、全体のバランスを考慮して伽羅間の戦闘能力の強さをOC管理の指示に沿って設定していただく必要があります。これは『機体』も同様であり、明確にCNA世界観に沿わない過剰なチートに関しては伽羅自体の却下、及びナーフ対象になり得ますのでご理解ください。

またモチーフとする出典元を重視とする都合上、特に『ARMORED CORE』シリーズや『機動戦士ガンダム』シリーズ、またリアル系ロボット作品に対するキャラクター・機体設定の優遇も発生致します。『CNA』の世界観ジャンルが近未来SFであることを踏まえて、ご理解いただけますと幸いです。

※ 仕様書の審査基準については別途ノート # Screening に記載していますのでそちらを参照してください。

ROLEPLAY 也（ロールプレイ）について
_₁_

当OCでは也の際、通常ロール（文言のみ）と特殊ロールで交遊形式を分けています。通常ロール利用者同士での交遊は『CNA: ロール１』に入室し「スレッドを立てて行ってください」。また期限の切れたスレッドの続きを行う場合、前スレッドのタイトル名を入力した上で「２」「３」と続番を銘を打って作成してください。

例 『（タイトル）』→『（タイトル）：２』

通常ロールでは原則、画像・絵文字・音声の使用は厳禁です。これらを行う際は交遊相手に事前確認を行った上、双方の同意の下で『CNA: ロール２』に入室して行ってください。

『CNA: ロール２』スレッド外では、自身の順番が終了後に末尾に「〆」や「END」のように括ってください。連投の占有時間の目安は15分までを目処とし、前ロールが終了したことが分かるまでは他の利用者様はロールを行わないでください。

_₂_

当OC全体でのロールにおける文字数下限（セリフを除いた描写・モノローグ）は「50文字」と定めます。ただしこれはあくまでOC内下限であり、交遊相手様の下限が50文字以上である可能性もあります。基本的にはロール前に予め背後部屋・エントランスで交遊相手様と相互確認をとることを推奨致します。

また所謂なりきり界隈での『空白厨』や『漢字厨』などは明確に「広義の通常ではない」と当OCでは定めます。ロールでの利用の際は原則、交遊相手様の確認を取ってください。

"読めないもの"は共通認識ではありません
これは厨の否定ではなく、場を弁えるということです。
_₃_

当OCではなりきりのローカルルールに倣い、原則、利用者様間の確炉留（確定ロール。相手の行動を自分のロールで確定させる行為）を非推奨にさせていただきます。あくまで「禁止」ではないのは、確炉留の定義を厳格にすると究極的には物語そのものが成立しなくならざるを得ない為です。交遊相手様との兼ね合い、また許容範囲を探る・確認し合う形で落とし込んでください。

またイベント時においてOC管理が確炉留を行うこともあります。これは全体のストーリー進行を踏まえ、物語展開があらぬ方向に向かう措置、ないしは物語品質が確炉留でも上がることがあると認識しているからです。無論それは無意味に利用者様を虐げることが目的ではなく、最終的な利用満足度を高める為にあります。

MISC その他

当OC内でのアイコン及びキャラクター仕様書の画像は原則「実写映画・ドラマ」のものに設定することを禁じます。これは肖像権を考慮したものであり、Blender（CGモデル）などによる画像、公式イラスト等の使用は可能です。また折伽羅の仕様書に用いる画像は実写以外であれば認可しますが、既存版権キャラクターの画像を用いることに関しては他利用者様の反感を買うことを念頭に入れてください。

問題発生または出典元との照合を込みで他利用者様からの申告があった場合は該当画像を却下させていただくことがあります。版権画像の利用に伴う責任につきましてはCNA全体・OC管理では負いかねますのでご注意ください。
